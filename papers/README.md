# Literature Notes: LLM-as-a-Judge

## Summary Template

Each paper is summarized using the following structure:

- **Paper title**
- **Key idea**
- **Judge models**
- **Dataset**
- **Prompt strategy**
- **Experiment setup**
- **Metrics**
- **Important findings**
- **Open questions**
